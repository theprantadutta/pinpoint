## Screenshot Guidelines

- **Resolution**: 1080x2400 (or your device's native resolution)
- **Format**: PNG (for transparency if needed)
- **Naming**: Use kebab-case (lowercase with hyphens)
- **Content**: Show the app in its best state with sample data
- **Theme**: Include both dark and light mode screenshots
- **Device**: Use consistent device frames if adding frames

## Tips

1. Use Android Studio's Device File Explorer or adb to capture screenshots
2. Consider using a tool like [Mockuphone](https://mockuphone.com/) to add device frames
3. Ensure text is readable and UI elements are clearly visible
4. Show off the glassmorphism effects and gradient backgrounds
5. Use consistent accent color across screenshots (Mint recommended)
